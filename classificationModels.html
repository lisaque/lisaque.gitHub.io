Comparing Classification Models using Python

In this project, I imported a database from Kaggle, which contains people's physical attributes: age, gender, weight, BMI (body-mass index) and a "Label" which displays if a person is "normal weight", "underweight", "overweight" or "obese".

I created my own KNN classification algorithm, then compared it to Python's built-in algorithms using sklearn. Then I compared how accurate each classification model was, in the classification of the dataset I used.
